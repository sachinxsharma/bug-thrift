# Hive & Swarm | Futuristic Bug Thrift Store 🪲

![Hero Banner](screenshots/hero-banner.png)

Welcome to **Hive & Swarm**, a futuristic e-commerce thrift store crafted specifically for the next era of the swarm—ants, flies, and bees looking to upgrade their exoskeletons. Browse our latest acquisitions including neon exoskeleton glasses, titanium honeycomb wings, and pure sweet blood, placing orders with the ultimate currency: drops of human blood.

## 🌟 Features
- **Interactive Theme Toggling:** Seamlessly switch between dark and light modes.
- **Dynamic Shopping Cart:** Add items to your tribute, edit quantities, and track your total blood cost.
- **Checkout Simulation:** Extract blood and place orders via an immersive UI/UX checkout flow.
- **Glassmorphism Design:** Modern, futuristic UI patterns tailored to a cyberpunk aesthetic.
- **Smooth Animations:** Hover states, glowing shadows, and modal transitions.

## 🛠️ Tech Stack
This project is built using vanilla web technologies, ensuring a lightweight footprint without the need for external build tools or frameworks.
- **HTML5:** Semantic architecture and meta tags.
- **CSS3 (Vanilla):** Custom CSS Variables, Flexbox/Grid Layouts, Glassmorphism, CSS Transitions, and animations.
- **JavaScript (ES6):** Vanilla DOM Manipulation, State Management (Cart Array), and Event Listeners.
- **Google Fonts:** Orbitron (futuristic headers) and Inter (body copy).
- **FontAwesome:** For rich iconography.

## 🚀 How to Clone and Run Efficiently

### 1. Clone the Repository
Clone this repository to your local machine using the Git CLI:
```bash
git clone <your-repository-url>
cd bug-thrift-store
```

### 2. Run the Application
Since this project uses vanilla HTML, CSS, and JS, there is no need for `npm install` or compilation steps. You can run it effortlessly using one of the following methods:

#### Method A: Python HTTP Server (Recommended)
If you have Python installed, you can start a local development server quickly:
```bash
python -m http.server 3000
```
Then navigate to `http://localhost:3000` in your web browser.

#### Method B: VS Code Live Server Extension
1. Open the project directory in Visual Studio Code.
2. Install the **Live Server** extension.
3. Right-click on `index.html` and select **"Open with Live Server"**.

#### Method C: Direct File Execution
You can easily drag and drop the `index.html` file right into any modern browser tab to run the app. However, running a local server (Methods A or B) is the standard best practice.
